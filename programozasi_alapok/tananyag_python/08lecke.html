Logikai változók, összehasonlító operátorok és logikai operátorok

Logikai értékek, változók.

Feladat:

Az interneten keressünk rá a logikai változókra!

Válaszok:

A logikai értékek (boolean) két érték egyikét képviselik: True vagy False.

Bármilyen kifejezést ki lehet értékelni a Pythonban, és megkaphatjuk a két válasz egyikét, True vagy False.

Ha két értéket összehasonlítunk, a kifejezés kiértékelődik, és a Python egy logikai választ ad vissza.
A tartalmazást vizsgáló operátor (membership operator) is boolean értékkel tér vissza.

Kiértékeléshez használhatjuk még a bool() függvényt, amely szintén vagy True vagy False.
Bármit odaadhatunk neki. Literált, változót, kifejezést stb.

Szinte minden True-ra értékelődik ki, kivéve a következőket. Minden, ami "üres"!
