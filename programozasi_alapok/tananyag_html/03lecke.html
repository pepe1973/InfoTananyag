Válaszok:

A OneCompiler egy ingyenes online fordító. Segíti a felhasználókat több mint 60 programozási nyelv és adatbázis kódjának online írásában, futtatásában és megosztásában.

Mi az a StackBlitz?

Feladat:

Az interneten keressünk rá a StackBlitz-re!

Válaszok:

A StackBlitz az együttműködésen alapuló böngészőalapú IDE webfejlesztők számára. A StackBlitz kiküszöböli az időigényes helyi konfigurációt, és lehetővé teszi, hogy közvetlenül a böngészőben írjon, futtasson kódokat és végezzen hibakeresést.

A StackBlitz egy online szerkesztő, amely az online IDE összes funkcióját biztosítja a Visual Studio Code, a parancssor és az alkalmazás előnézeti módjának összekapcsolásával.

Mi az a Visual Studio (VS) Code for the Web?

Feladat:

Az interneten keressünk rá a Visual Studio (VS) Code for the Web-re!

Válaszok:

A Visual Studio Code for the Web ingyenes, telepítés nélküli Microsoft Visual Studio Code élményt biztosít, amely teljes egészében az Ön böngészőjében fut, és lehetővé teszi a forráskód-tárolók gyors és biztonságos böngészését, valamint könnyű kódmódosításokat.

A VS Code for the Web a VS Code Desktop számos olyan funkciójával rendelkezik, amelyeket szeret, beleértve a keresést és a szintaktikai kiemelést böngészés és szerkesztés közben, valamint a bővítmények támogatását a kódbázison való munkához és az egyszerűbb szerkesztésekhez.
A tárolók, elágazások és a forrásvezérlő szolgáltatóktól, például a GitHubtól és az Azure Repostól származó lekérések megnyitása mellett a helyi gépen tárolt kóddal is dolgozhat.
